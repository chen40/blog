项目开始时间：10月23日
项目预计结束：12月23日
项目名称：个人博客
项目负责任：陈世林
项目内容：首页展示（主目录，关于，从事，产品，人际关系）
项目环境：django+Html+MySQL

过程：
	1.创建博客项目
		django-admin startproject blog
	2.创建首页应用
		python manage.py startapp index
	3.在应用中创建templates和static目录
	4.更改配置文件
		1）添加应用
		2）数据库配置
		3) 语言和时区
		4）静态文件存放路径
	5.编写主目录urls
		1）导入include模块
		2）添加应用的urls
	6.创建应用的urls.py文件，并添加：
		from django.conf.urls import url
		from .views import *
		urlpatterns = [(), (), ]
	7.完善views功能
	8.启动服务 python manage.py runserver
		报错：ModuleNotFoundError: No module named 'MySQLdb'
		原因：mysql不支持python3.x
		解决办法：在项目的__init__.py中添加
			import pymysql
			pymysql.install_as_MySQLdb()
